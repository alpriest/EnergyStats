Get an instant overview of how your FoxESS PV/Battery installation is performing!

- If you have a battery you can see duration until full/empty
- Automatically re-login
- Graphs of inverter parameters
- Historic statistics information
- Show battery temperature
- Modify battery charge times
- Modify minimum SOC values
- Modify inverter work schedule
- Shortcuts to let you retrieve battery SOC

This app requires you to have an account with FoxESS Cloud and API Key in order to login.
